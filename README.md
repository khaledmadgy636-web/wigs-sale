# Wigs Sale — موقع البيع

موقع هبوط (Landing Page) لبيع باروكات الشعر الطبيعي.

## النشر على Vercel

1. ارفعي محتويات هذا المجلد إلى مستودع (repository) على GitHub.
2. ادخلي على [vercel.com](https://vercel.com) وسجّلي بحساب GitHub.
3. اضغطي **Add New → Project** واختاري المستودع.
4. Vercel يكتشف أنه موقع ثابت (Static) تلقائياً — اضغطي **Deploy** مباشرة.
5. بعد دقائق يطلع لك رابط مباشر (مثل `your-site.vercel.app`).

## البنية

```
.
├── index.html        ← الصفحة الرئيسية (خفيفة، تفتح فوراً)
├── images/           ← كل الصور (75 صورة)
└── vercel.json       ← إعدادات الكاش والأداء
```

> ⚠️ لا تفصلي مجلد `images` عن `index.html` — يجب أن يبقيا معاً.

## ملاحظات الأداء

- ملف `index.html` خفيف (~205KB) فيفتح فوراً بدون شاشة انتظار.
- الصور منفصلة وتُخزّن في كاش المتصفح سنة كاملة (`max-age=31536000`) فتحمّل مرة واحدة فقط.
